
# 🚀 Margdarshan AI – Career Recommender for Tier 2/3 Students

🔗 **Live App:** [https://nimish-codes-exe-margdarshan-ai-app-mi88ku.streamlit.app/](https://nimish-codes-exe-margdarshan-ai-app-mi88ku.streamlit.app/)

---

## 📌 Overview

**Margdarshan AI** is an intelligent career recommendation system designed specifically for **Tier 2 and Tier 3 students**.

It helps users identify suitable career paths based on their **skills, interests, and background**, while also providing **structured roadmaps** to achieve those careers.

Built using **Streamlit**, this project transforms a Python-based ML/logic system into an interactive web application without requiring complex frontend development ([GeeksforGeeks][1]).

---

## 🎯 Problem Statement

Students from Tier 2/3 colleges often face:

* Lack of proper career guidance
* Confusion between multiple career paths
* No clear roadmap to reach their goals
* Limited access to mentors and structured planning

**Margdarshan AI solves this by acting as a personalized AI mentor.**

## 🏗️ Architecture

```
User Input → Processing Engine → Recommendation Model → Output UI
```

### Flow:

1. User enters skills/interests
2. System processes input using similarity/ML logic
3. Matches with predefined career datasets
4. Generates:

   * Career suggestions
   * Roadmap
5. Displays results via Streamlit UI
